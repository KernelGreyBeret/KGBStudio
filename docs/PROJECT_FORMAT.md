# KGB Studio Project Format

KGB Studio projects should stay readable JSON.

Preferred extension:

`.kgbproject.json`

Core shape:

```json
{
  "meta": {},
  "mission": {},
  "assets": {
    "sprites": [],
    "animations": [],
    "shapes": [],
    "particles": [],
    "sounds": [],
    "tiles": [],
    "palettes": [],
    "maps": []
  },
  "settings": {}
}
```
